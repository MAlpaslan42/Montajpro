# MontajPro – Android'e Kurulum Rehberi

## Adım 1: GitHub hesabı aç (ücretsiz)
1. github.com adresine gidin
2. "Sign up" ile ücretsiz hesap açın

## Adım 2: Yeni repository oluştur
1. Sağ üstte "+" → "New repository"
2. Repository name: montajpro
3. "Public" seçin
4. "Create repository" tıklayın

## Adım 3: Dosyaları yükle
1. "uploading an existing file" linkine tıklayın
2. Şu 4 dosyayı sürükleyip bırakın:
   - index.html
   - manifest.json
   - sw.js
   - icon.svg
3. "Commit changes" tıklayın

## Adım 4: GitHub Pages aç
1. Repository'de "Settings" sekmesi
2. Sol menüde "Pages"
3. Source: "Deploy from a branch"
4. Branch: main → / (root)
5. "Save" tıklayın
6. Birkaç dakika bekleyin → adresiniz hazır:
   https://KULLANICIADINIZ.github.io/montajpro

## Adım 5: Android'e kur
1. Chrome'da adresinizi açın
2. Otomatik "Kur" banner'ı çıkar → "Kur" tıklayın
3. Veya Chrome menü (⋮) → "Ana ekrana ekle"
4. ✅ Uygulama kuruldu!

## Önemli Notlar
- İnternet bağlantısı olmadan da çalışır (offline)
- Veriler telefonunuzda güvenle saklanır
- Ücretsiz, reklam yok
- Güncelleme için sadece dosyaları tekrar yükleyin
